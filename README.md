# ClayTech

Projeto web profissional desenvolvido como **projeto extensionista da graduação em Análise e Desenvolvimento de Sistemas**, com foco na criação de uma presença digital moderna, responsiva e profissional para um **cliente real do segmento de soluções em informática**.

🔗 **Acesse o site:** https://claytech.vercel.app/  

---

## 📌 Sobre o projeto

O **ClayTech** foi desenvolvido com o objetivo de aplicar, em um cenário real, conceitos fundamentais de desenvolvimento front-end, design responsivo e experiência do usuário.

Diferente de um projeto puramente acadêmico, este trabalho foi construído a partir de uma **demanda real de cliente**, aplicando o fluxo de trabalho encontrado no mercado de tecnologia.

A proposta foi transformar uma necessidade concreta de negócio em uma solução digital funcional, moderna e escalável, conectando teoria e prática.

O foco principal foi entregar uma aplicação com:

- interface moderna e profissional
- navegação intuitiva
- layout totalmente responsivo
- boa performance
- código limpo e organizado
- estrutura escalável para manutenção futura

---

## 🚀 Tecnologias utilizadas

Este projeto foi desenvolvido utilizando as seguintes tecnologias:

- **ReactJS**
- **JavaScript**
- **Vite**
- **Tailwind CSS**
- **HTML5**
- **CSS3**
- **Vercel (Deploy)**

Durante o desenvolvimento, os principais objetivos técnicos foram:

- foco em usabilidade e experiência do usuário

Durante o processo, foram trabalhadas competências práticas importantes para atuação profissional, como:

- levantamento de requisitos
- entendimento da necessidade do cliente
- definição da melhor abordagem técnica
- construção da interface
- testes visuais e ajustes

Essa experiência contribuiu diretamente para o aprimoramento do meu conhecimento técnico e da minha visão de desenvolvimento aplicado ao mercado.
